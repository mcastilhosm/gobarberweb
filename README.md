## 💻 Começando

**Clone o projeto e acesse a pasta**

```bash
$ git clone https://github.com/mcastilhosm/gosoccer.git && cd gosoccer
```

**Siga os passo abaixo**

```bash
# Instale as dependências
$ yarn

# Inicie
$ yarn start
```
